## 📄 Description  
Space Explorer is an interactive web application that allows users to explore images provided by the NASA API and save their favorites in a personal gallery. The frontend handles the user interface and communicates with the backend to deliver a responsive user experience.

## 🚀 Features  
- Explore daily astronomical images from NASA’s API.  
- User authentication and route protection integrated with the backend.  
- Save, view, and delete favorite images in a personalized gallery.  
- Responsive UI optimized for various screen sizes.

## 🛠️ Technologies  
- **Vite**: A fast development building tool. 
- **JavaScript**: Adds logic, behavior, and interactivity to the application. 
- **CSS3 + Media Queries**: Used to style the interface and ensure responsiveness across devices.  
- **React**: A library for building modular and reusable UI components.  
- **React Router**: Handles client-side navigation.  
- **React Hook Form**: A lightweight library for managing and validating forms.  
- **Context API**: Provides global state management for sharing data across components.
